* 개요

NixOS tmux 설정 키바인딩 참조 문서.
설정 위치: =users/junghan/modules/shell.nix=

Prefix: =Ctrl-a= (screen 스타일)

* Copy/Paste (vi-copy 모드)

** Copy 모드 진입/종료

| 키        | 동작              |
|-----------+-------------------|
| =C-a [=   | Copy 모드 진입    |
| =q=       | Copy 모드 종료    |
| =Escape=  | Copy 모드 종료    |

** 선택 및 복사

| 키        | 동작                              |
|-----------+-----------------------------------|
| =v=       | 선택 시작                         |
| =C-v=     | 사각형 선택 토글                  |
| =y=       | 복사 + 시스템 클립보드 + 모드 종료 |
| =Enter=   | 복사 + 시스템 클립보드 + 모드 종료 |

** 붙여넣기

| 키        | 동작                         |
|-----------+------------------------------|
| =C-a ]=   | 시스템 클립보드에서 붙여넣기 |

** 이동 (vi 스타일)

| 키           | 동작            |
|--------------+-----------------|
| =h j k l=    | 커서 이동       |
| =w / b=      | 단어 단위 이동  |
| =0 / $=      | 줄 시작/끝      |
| =g / G=      | 문서 시작/끝    |
| =C-u / C-d=  | 반 페이지 이동  |
| =C-b / C-f=  | 전체 페이지 이동 |

** 검색

| 키    | 동작          |
|-------+---------------|
| =/=   | 아래로 검색   |
| =?=   | 위로 검색     |
| =n=   | 다음 검색결과 |
| =N=   | 이전 검색결과 |

* 윈도우 관리

| 키        | 동작                       |
|-----------+----------------------------|
| =C-a c=   | 새 윈도우 (현재 경로)       |
| =C-a ,=   | 윈도우 이름 변경           |
| =C-a w=   | 윈도우 목록                |
| =C-a n=   | 다음 윈도우                |
| =C-a p=   | 이전 윈도우                |
| =C-a 0-9= | 윈도우 번호로 이동         |
| =C-a &=   | 윈도우 종료 (확인)          |

* 패널(Pane) 관리

** 분할

| 키        | 동작                      |
|-----------+---------------------------|
| =C-a |=   | 세로 분할 (현재 경로)     |
| =C-a -=   | 가로 분할 (현재 경로)     |

** 이동

| 키        | 동작       |
|-----------+------------|
| =C-a h=   | 왼쪽 패널  |
| =C-a j=   | 아래 패널  |
| =C-a k=   | 위 패널    |
| =C-a l=   | 오른쪽 패널 |

** 크기 조절

| 키            | 동작           |
|---------------+----------------|
| =Alt-Left=    | 왼쪽으로 5칸   |
| =Alt-Right=   | 오른쪽으로 5칸 |
| =Alt-Up=      | 위로 5칸       |
| =Alt-Down=    | 아래로 5칸     |

** 기타

| 키        | 동작                |
|-----------+---------------------|
| =C-a x=   | 패널 종료 (확인)    |
| =C-a z=   | 패널 확대/축소 토글 |
| =C-a !=   | 패널을 윈도우로 분리 |
| =C-a q=   | 패널 번호 표시      |

* 세션 관리

| 키        | 동작           |
|-----------+----------------|
| =C-a d=   | 세션 분리      |
| =C-a s=   | 세션 목록      |
| =C-a $=   | 세션 이름 변경 |
| =C-a (=   | 이전 세션      |
| =C-a )=   | 다음 세션      |

* 유틸리티

| 키        | 동작                 |
|-----------+----------------------|
| =C-a r=   | 설정 파일 리로드     |
| =C-a ?=   | 키바인딩 목록        |
| =C-a :=   | 명령 프롬프트        |
| =C-a t=   | 시계 표시            |

* 마우스 지원

- 마우스로 패널 선택
- 마우스로 패널 크기 조절
- 마우스 드래그로 텍스트 선택 → 자동 복사

* 설정 옵션

| 옵션                   | 값           | 설명                    |
|------------------------+--------------+-------------------------|
| =base-index=           | 1            | 윈도우 번호 1부터 시작  |
| =escape-time=          | 0            | Escape 딜레이 없음      |
| =history-limit=        | 10000        | 스크롤백 히스토리       |
| =mouse=                | on           | 마우스 지원             |
| =mode-keys=            | vi           | vi 스타일 키바인딩      |
| =automatic-rename=     | off          | 자동 이름변경 끔        |
| =renumber-windows=     | on           | 윈도우 번호 자동 재정렬 |

* CLI 명령어

#+begin_src bash
# 세션 생성
tmux new -s mysession

# 세션 목록
tmux ls

# 세션 연결
tmux attach -t mysession

# 세션 종료
tmux kill-session -t mysession

# 모든 세션 종료
tmux kill-server
#+end_src
